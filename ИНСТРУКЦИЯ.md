# Инструкция по запуску VK-бота

## Шаг 1 — Узнай свой VK ID (для уведомлений о лидах)

1. Зайди на vk.com
2. В адресной строке твоей страницы — числа после /id (например vk.com/id12345678)
3. Если адрес короткий (vk.com/ivanov) — зайди на vk.com/id0 и система перенаправит на твой профиль

## Шаг 2 — Заполни config.py

Открой `config.py` и замени значения:
- `ВСТАВЬ_ТОКЕН_СЮДА` → токен группы, который ты уже получил
- `0` в ADMIN_VK_ID → твой числовой VK ID
- GROUP_ID уже заполнен: 239389077

Пример готового config.py:
```python
VK_TOKEN = "vk1.a.WfhUFdQ9..."   # твой токен
ADMIN_VK_ID = 123456789           # твой VK ID
GROUP_ID = 239389077              # ID группы
```

## Шаг 3 — Замени партнёрскую ссылку в bot.py

Найди строку с комментарием `# TODO: замени на свою партнёрскую ссылку`
и замени:
```
https://eda.yandex.ru/couriers
```
на свою реальную партнёрскую ссылку Яндекс Еды.

## Шаг 4 — Деплой на Railway (рекомендуется)

Это тот же процесс, что и для Telegram-бота:

1. Создай новый репозиторий GitHub → загрузи три файла:
   - `bot.py`
   - `config.py`
   - `requirements.txt`

2. Зайди на railway.app → New Project → Deploy from GitHub
3. Выбери репозиторий с VK-ботом

4. В разделе **Variables** добавь:
   - `VK_TOKEN` = токен группы
   - `ADMIN_VK_ID` = твой VK ID
   - `GROUP_ID` = 239389077

5. В **Settings → Custom Start Command** укажи:
   ```
   python bot.py
   ```

6. Deploy — бот запустится!

## Шаг 5 — Проверка

1. Зайди в свою группу ВКонтакте
2. Напиши в сообщения группы «Привет» или «/start»
3. Бот должен ответить приветственным сообщением с кнопками
4. Пройди весь сценарий — в конце тебе придёт уведомление в личку ВКонтакте

## Возможные проблемы

| Проблема | Решение |
|---|---|
| Бот не отвечает | Проверь, включены ли сообщения в группе |
| Ошибка токена | Убедись, что токен скопирован полностью |
| Нет уведомлений | Проверь ADMIN_VK_ID — это твой числовой ID |
| `longpoll error` | Проверь, что Long Poll API включён в настройках группы |

## Важно — включи сообщения в группе!

Управление группой → Сообщения → включи тумблер
Иначе пользователи не смогут написать боту.
